Detailed Description

template<typename ETYPE>
class OODictionary< ETYPE >

NSMutableDictionary wrapper for use in instance variables which allows subscripting by the key value. Use &~dict["key"] to remove the entry and return it with expression or autorelease scope. Subscripting can be applied recusively. ETYPE is type of leaf node.

Operators:

operatorinplacearguments
take referernce=dictionary
take copy<<=dictionary
merge entries+=dictionary
remove entries-=key or array
sbuscript[]object (key) or "slice" of keys

Usage: 

    

  • (void)function:(OODictionary<NSString *> &)dict {
    dict["ONE"][@"TWO"][@"THREE"] = @"Four"; // set valule
    NSString *str = *dict[@"ONE"][@"TWO"][@"THREE"]; // get value
    ~dict[@"ONE"][@"TWO"][@"THREE"]; // delete value
pointer "str" is invalid at this point
 }
